TPhotos
=========

Please check http://pas66430.free.fr/TPhotos for more.
Install or update Java (minimal version 8).

Which file to use?
==================

Linux:
- Debian users : TPhotos.deb
- RPM based users : TPhotos.rpm
- All others Linux users : TPhotos.zip
(unzip this file into your TPhotos folder)

MacOS : TPhotos.tar.gz
(extract this content into your TPhotos folder)

Windows users : TPhotos.exe

Project source code : TPhotos_src.zip
